# Python: Da Zero a Robot & Web Bot

Percorso di studio semestrale per imparare Python da zero,
con focus su robotica (robot/cobot) e web bot.

## Struttura

- **Mese 1** (Sett. 1-4): Fondamenta di Python
- **Mese 2** (Sett. 5-8): Strutture Dati e File
- **Mese 3** (Sett. 9-12): OOP e Librerie Esterne
- **Mese 4** (Sett. 13-16): Web Bot e Automazione
- **Mese 5** (Sett. 17-20): Robotica e Cobot
- **Mese 6** (Sett. 21-24): Integrazione e Progetto Finale
